========================================================================
Tamagotchi
========================================================================

Cross platform OpenGL ES 2.0 game engine for PC and iOS written in C++.


Build
------------------------------------------------------------------------
Requirements: Visual Studio 2012 or XCode 4+.
3rd party library .lib files are compiled with VS2012 and .a files with Clang 4.2
